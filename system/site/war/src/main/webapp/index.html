Cubeia Poker (Open Source)

Standalone Single Server Installation

Register: Create a Profile

Play : Go to the Game

Website

http://localhost:8181

A very basic website (this page)

User Registration

http://localhost:8181/register.html

A very basic user registration (register before log in)

Poker Client

http://localhost:8181/game

Join a table. Use a 2nd browser for a 2nd player to make a quick test.

Bots Admin

http://localhost:8081

Set 10 or 50 bots and press the OK button. You should see within the lobby that the tables are getting players. Join one or more tables and check the game.

After this, you can start tournament bots, asking them to leave one seat open, so you can join a tournament.

IMPORTANT: Bots are like other users, so if you want to use the bots, you have to create user-accounts for them, too. Example for 200 bots (Bot_1 ... Bot_200):

  cd tools/user-creator/target
  java -jar user-creator.jar -d -u Bot -p 1 -o 0 -r 200

Poker Admin

http://localhost:8088

Create a Table Template, which should result in tables showing up in the lobby. The names for the tables are picked up from a list.

Now you can create a simple Sit&Go, which should show up in the lobby.

Last, create a Scheduled Tournament template.

That's it, the Cubeia Poker System is running.
